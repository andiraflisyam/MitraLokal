**MitraLokal** adalah platform berbasis web yang dirancang untuk membantu Usaha Mikro, Kecil, dan Menengah (UMKM) dalam memperluas jangkauan pasar secara digital. Tidak hanya berfungsi sebagai katalog produk dan media transaksi, platform ini juga menyediakan modul edukasi untuk membantu pelaku usaha mengembangkan bisnis mereka.

## ğŸ’» Cara Menjalankan Project (Installation)

Ikuti langkah ini untuk menjalankan project di komputer lokal (Localhost):

1.  **Clone Repositori**
    ```bash
    git clone [https://github.com/username-kamu/mitralokal.git](https://github.com/username-kamu/mitralokal.git)
    ```
2.  **Persiapan Database**
    * Buka XAMPP/WAMP, nyalakan Apache dan MySQL.
    * Buka `localhost/phpmyadmin`.
    * Buat database baru dengan nama `db_mitralokal`.
    * Import file `db_mitralokal.sql` yang ada di folder database project ini.
3.  **Konfigurasi**
    * Buka file `config.php` (atau file koneksi database kamu).
    * Sesuaikan username dan password database jika diperlukan.
4.  **Jalankan**
    * Pindahkan folder project ke `htdocs`.
    * Akses melalui browser: `localhost/mitralokal`.
